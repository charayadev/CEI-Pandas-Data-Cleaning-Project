## 🏢 Internship Overview

This repository documents all assignments and projects completed during the **Celebal Excellence Internship (CEI)** — a structured, industry-aligned internship programme focused on building real-world skills in data analytics, data engineering, SQL, and Python.

Each week introduces new concepts through hands-on, project-based work designed to simulate professional data workflows.

## 📖 About This Repository

**CEI-Assignments** is a week-wise collection of practical work completed throughout the internship. Each folder represents one week's assignment — covering a distinct domain or tool — and includes source code, queries, and any supporting files.

The work here demonstrates applied proficiency in:
- Cleaning and transforming raw data with Python and Pandas
- Writing structured SQL queries for real-world business problems
- Exploring relational databases and applying database design concepts
- Building analytical thinking through incremental, project-based tasks

## 📂 Detailed Project Descriptions

---

### 📌 Week 01 — Pandas Data Cleaning and Analysis

> **Domain:** Data Cleaning & Exploratory Data Analysis
> **Language:** Python

**Overview:**
Applied Python and the Pandas library to clean and prepare a raw CSV dataset for analysis. The project covered the full data preparation pipeline — from identifying and handling missing values to feature engineering and exploratory analysis.

**Skills Demonstrated:**
- Loading and inspecting raw CSV data
- Identifying and handling missing values
- Removing duplicate records
- Data type correction and normalisation
- Feature engineering from existing columns
- Exploratory data analysis (EDA)

---

### 📌 Week 02 — SQL Sales Analysis Project

> **Domain:** Relational Database Analysis
> **Database:** PostgreSQL | **Tool:** pgAdmin 4

**Overview:**
Loaded a sales dataset into a PostgreSQL database and performed 27 structured SQL queries covering the full range of SQL operations — from basic filtering to multi-table joins, transactions, and ACID compliance.

**Objectives Completed:**
- Loaded dataset into PostgreSQL and explored schema and tables
- Applied filtering using `WHERE`, `BETWEEN`, `IN`, and `LIKE`
- Performed aggregations with `GROUP BY`, `HAVING`, and `ORDER BY`
- Extracted business insights from customer and product data
- Joined multiple tables using `INNER JOIN` and `LEFT JOIN`
- Validated data quality through constraint-based queries
- Implemented `CASE` statements for conditional categorisation
- Practised transactions using `BEGIN`, `COMMIT`, and `ROLLBACK`
- Explored ACID properties and foreign key relationships

**SQL Concepts Covered:**

`SELECT` · `WHERE` · `GROUP BY` · `HAVING` · `ORDER BY` · `INNER JOIN` · `LEFT JOIN` · `CASE` · `Aggregations` · `Constraints` · `Transactions` · `ACID Properties` · `Data Validation`
